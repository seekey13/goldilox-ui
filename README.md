# Goldilox

Goldilox is an Ashita4 addon exclusively for the [CatsEye XI](https://www.catseyexi.com/) custom daily content 
consisting of goblin dailies, including the goblin by the same name.

## Usage

Load Goldilox with `/addon goldilox`.  With the addon loaded, talk to goblins to track what your dailies are.  
Leave the addon loaded while working on said dailies (i.e. killing required creatures or finding the secret chest).

Typing `/goldilox` or `/dailies` will report the status of all current dailies.  Daily status resets at JP Midnight.
