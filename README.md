# Goldilox-UI

[Goldilox](https://github.com/dewiniaid/ffxi-ashita-goldilox) is an Ashita4 addon exclusively for the [CatsEye XI](https://www.catseyexi.com/) custom daily content 
consisting of goblin dailies, including the goblin by the same name.  This was originally developed by [dewiniaid](https://github.com/dewiniaid/ffxi-ashita-goldilox/commits?author=dewiniaid)


Goldilox-UI is my spin on it to consolidate the information in an imgui panel to make the process more visual.

## UI

`/huh motion` button will issue the command then parse the positional text and display `Far East` in the panel for example

`#sea` is a simple hyperlink to the Sea channel on the CatsEyeXI Discord where Flux locations are discussed.

All the other `Green Text Links` will navigate you to the [FFXI Wiki](https://www.bg-wiki.com/ffxi/) page for additional information.  Quick way to figure things out, if you are not familar with the Mob/Zone/Item.

## Usage

Load Goldilox-UI with `/addon load goldilox-ui`.  With the addon loaded, talk to Goblins and Palalumin to track what your dailies are.  If you are grouping up be sure to Party before talking to NPCs to ensure the quests align.

Leave the addon loaded while working on said dailies (i.e. killing required creatures or finding the secret chest).

Daily status resets at JP Midnight.
